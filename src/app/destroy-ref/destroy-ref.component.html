1. Description
In Angular 16, it provides a flexible and injectable way to do clean up or destruction. Instead of using the life function ngOnDestroy, we can inject DestroyRef and reister the onDestroy lifecycle hook.
Compared to the life function ngOnDestroy, it has a few properties.
  • DestroyRef is injectable, we can create a reusable function and put our cleanup logic in
  • ngOnDestroy lifecycle hook method would run before DestroyRef.onDestroy() if we had both of them in the same scope
