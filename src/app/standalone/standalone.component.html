1. Description
Angular provides standalone APIs from Angular v14 (Developer preview) and Angular v15 .
In v16, there are a few new enhancements for standalone APIs.
  1. Command to transform the legacy code to the stanalone APP automatically (Migration guide)
  2. Command to generate the stanalone APP
  3. Configuration of NgZone for standalone APP
2. ngZoneEventCoalescing
In the default strategy or the OnPush strategy, the button clicking will triiger the change detection of NgZone. Even the event was propagated by the event bubbling mechanism, the change detection will still happen.
After Angular 9, we could coalesce events together so that there is only one change detection.
